OutOfCoffeeException Angular Components

This page demonstrates the oocx-ng Angular components. For more information and source code, visit the oocx-ng homepage on github. You can download these components from npmjs.

Components

oocx-ng contains a collection of lightweight Angular components:

Calendar

The calendar component shows a calendar with German day / month names and week numbers. The user can select a date and can navigate to the previous / next month.

Date Input

The date input combines a date autocomplete service with the calendar component. The user can imput the date or select a date in the calendar component.

Number Input

A textbox that limits input to numbers.

Autocomplete

A text input with autocomplete popup. The autocomplete entries can be simple text (default) or complex content (by specifiing a custom template).

Autocomplete Multiselect

This control works similar to the autocomplete component, but allows the selection of multiple values from the autocomplete popup.

Selection-List

Offers a list of choices which are presented as buttons.

Checkbox

The checkbox component combines an input of type 'checkbox' with a label component. The checkbox can be checked / unchecked by clicking on the label. IDs for the input and the label for-attribute are generated automatically.
Checkbox

Label

The label component can be used to generate a HTML label element for an input element. The component automatically generates a unique ID for the input element and uses this ID for the for-attribute of the label.

Spinner

The spinner component can be used as loading animation. It can subscribe to an observable and can display an error message if the observable throws an error.
